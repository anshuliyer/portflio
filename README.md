# ENGINEIYER.OS | Portfolio

A retro game console-themed portfolio for an Embedded Systems Engineer. Built with Vanilla JS, CSS, and HTML.

## 📟 Features
- **Retro CRT Aesthetic**: Scanlines, screen flicker, and barrel distortion.
- **Procedural Audio**: Web Audio API-driven sound effects for booting and navigation.
- **Interactive background**: Pulse and glow effects on circuit traces.
- **Boot Sequence**: Cinematic BIOS-style loading screen.
- **Touch Friendly**: Optimized for both desktop and mobile devices.

## 🎮 Navigation
- **Keyboard**: Use `UP/DOWN` or `W/S` to navigate, `ENTER` to select, and `ESC` to return to menu.
- **Touch**: Tap menu items to select.

## 🚀 Local Setup
1. Clone the repository.
2. Run a local server:
   ```bash
   python3 -m http.server 8000
   ```
3. Visit `http://localhost:8000`.

## 🛠 Tech Stack
- HTML5 / CSS3 / JavaScript
- Web Audio API
- Canvas API (for Background Dynamics)
